What are Colab Notebooks?

Google Colab is an environment that allows you to create an editable page using simplified Python code. You can use Colabs to display and plug values into equations, create visualizations, and to insert descriptive text. Using actions like “plot” or “import,” you can create an editable page without having to do the more complex coding behind each of those tasks, which makes Colab Notebooks a very friendly environment for working with Python. Below, you'll find a video walkthrough for each Colab notebook we'll use in the course, as well as a link to open the Colab in a new tab.
