What are Jupyter Notebooks?

Jupyter Notebook is a program that allows you to create an editable page using simplified code. They can be used to display and plug values into equations, create visualizations, and to insert narrative text. Using actions like “plot” or “import”, you can create your editable page without having to do the more complex html coding behind each of those actions, which makes Jupyter Notebooks an easy to use coding tool.

Jupyter Notebooks have 3 essential components. The first is live code, which is divided into individual boxes. “Live” just means you can edit any part of the code and re-run the whole program or just that one line to change the page. The second component is visualization, which means you can embed charts, graphs, images and other visual elements into your Notebook. The last component is rich text, which is just normal text you use in common writing programs, such as Microsoft word (also like the text you’re reading on this webpage now).
