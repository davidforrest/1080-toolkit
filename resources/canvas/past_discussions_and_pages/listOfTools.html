Tools

REQUIRED:

  1. Audacity: http://www.audacityteam.org
  2. Python (through Jupyter Notebooks and ThinkDSP library)
  3. iAudio Tool for iPhone: https://itunes.apple.com/us/app/iaudiotool/id580740399?mt=8 or Android: https://play.google.com/store/apps/details?id=com.julian.apps.AudioTool&hl=en  and user guide here (https://sites.google.com/site/bofinit/iaudiotool)

OPTIONAL:

  1. Programming and mathematical analysis:
    1. Python (local install) -- our computational lab can assist in setting up a local install of Python for you if you want to explore.
    2. Matlab (from FAS): http://downloads.fas.harvard.edu/download, Audio Signal Processing Toolbox: https://www.mathworks.com/products/audio-system.html
    3. Comsol: multiphysics finite element software
    4. Mathematica (from FAS):  http://downloads.fas.harvard.edu/download, http://www.wolframalpha.com/examples/Music.html
  2. Audio visualization:
    1. RTgram -- a real-time spectrogram display (can’t read files): http://www.phon.ucl.ac.uk/resource/sfs/rtgram/  
    2. Wavesurfer: https://sourceforge.net/projects/wavesurfer/
    3. Sonic Visualizer: http://www.sonicvisualiser.org , http://www.sonicvisualiser.org/videos.html
  3. Audio production software:
    1. Pro Tools
    2. Logic
    3. Garage Band
  4. Notation and composition tools:
    1. Finale ($$)
    2. Sibelius ($$)
    3. Musescore (free)
  5. Music programming:
    1. Max: visual programming language for audio processing https://cycling74.com/products/max/#.WOZhEBLyvdQ, https://cycling74.com/articles/find-a-max-online-class/
    2. Pure Data (Pd) software: https://puredata.info/docs/tutorials (Similar to Max see here for example: https://www.youtube.com/watch?v=DgeTHuDSgC0)
    3. Martingale: Pedagogical Tools in PD: https://code.google.com/archive/p/martingale/ Martingale provides a high-level library of cross-platform Pure Data (Pd) abstractions for quickly and easily creating complex real-time instruments and signal processors.
    4. Csound (computer programming language for sound): https://sourceforge.net/projects/csound/
    5. SuperCollider (Audio synthesis and composition): http://supercollider.github.io Analysis, synthesis, and processing of audio signals, uses a programming language Tutorial: http://doc.sccode.org/Tutorials/Getting-Started/00-Getting-Started-With-SC.html
  6. Audio samples:
    1. Free sound Project, free sound effect downloads: http://www.makeuseof.com/tag/freesound-free-sound-effects-download/
    2. Useful free sound files: https://freesound.org/people/xserra/packs/13038/
